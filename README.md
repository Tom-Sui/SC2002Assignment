<h1>===Steps to push codes===</h1>

<h2>Initialize</h2>

```
git init
```

<h2>Branches</h2>

To switch to a branch:
```
git checkout BRANCH_NAME
```

To create a new branch:
```
git checkout -b BRANCH_NAME
```

<h2>Download Code</h2>

If first time download

```
git clone HTTP
```
```
cd SC2002Assignment
```

If update code

```
git pull
```

<h2>Upload Code</h2>
Ensure correct branch by:

```
git branch
```
To upload your code:
```
git push
```
